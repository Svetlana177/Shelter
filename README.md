# shelter

Посетить сайт https://svetlana177.github.io/shelter/



#How To Run Locally
1. clone repo with:
```
git clone git@github.com:Svetlana177/shelter.git
```
2. go into the project folder with:
```
cd shelter/
```
3. install all dependencies with:
```
npm install
```
4. run in the **dev** mode with:
```
npm start
```
